# 🌱 Crop Recommendation System using Machine Learning and Django

The Crop Recommendation System is a machine learning based web application that helps farmers or users identify the most suitable crop to cultivate based on soil nutrients and environmental conditions.

The system takes user inputs such as Nitrogen, Phosphorus, Potassium, Temperature, Humidity, pH, and Rainfall, and predicts the **top 3 recommended crops** using a trained machine learning model.

This project also includes user authentication using Django (Registration and Login) and stores users in a MySQL database.

---

## 🚀 Features
- User Registration and Login Authentication
- Secure Django Authentication System
- Crop Prediction using Machine Learning
- Top 3 Crop Recommendations with Probabilities
- Interactive Web Interface using Django
- MySQL Database Integration
- User-Friendly Dashboard

---

## 🧠 Machine Learning Model
This project uses:

- Random Forest Classifier (trained on crop recommendation dataset)

### Input Features
- Nitrogen (N)
- Phosphorus (P)
- Potassium (K)
- Temperature
- Humidity
- pH
- Rainfall

### Output
- Top 3 Recommended Crops
